# Board Game Playtesting

A tool for organizing and tracking board game playtesting sessions.

## Overview

This project helps game designers manage their playtesting process, including:

- **Session Tracking** - Log playtesting sessions with notes and feedback
- **Player Management** - Keep track of playtesters and their feedback
- **Version Control** - Track different versions/iterations of game rules
- **Feedback Analysis** - Organize and analyze player feedback
- **Balance Tracking** - Monitor game balance metrics across sessions
